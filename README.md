# The Arena Hub Ltd – Clinical Evidence Infrastructure

**The Arena Hub Ltd (#1708605)** builds sovereign hybrid infrastructure for UK AP/SEMH schools. Arena v7.0 automates Pearson FS evidence, forensic safeguarding (VAR Sentry), and Ofsted portfolios—cutting teacher admin 90% (6.5 → 0.5 hrs/wk).

## 🚀 Live Pilot
- **3 Pillars**: Performance Centre (velocity tracking), Welfare Suite (forensic audit), Kit Room (AI vocational mapping).
- **Tech**: React UI → Cloud Run ingestor → GAS Ledgers.
- **Pilots Open**: [j.baguley@thearenahub.co.uk](mailto:j.baguley@thearenahub.co.uk)
